SIP providers tested by our users


In general, any SIP provider should work. If you’re using Telephone with a provider not listed here, please share your settings in our mailing list or send them directly to me.

Please note, if you’d like to have multiple accounts and there are different STUN servers provided, you can choose any STUN server, it does not matter which one. STUN is just a technology used to discover external IP address from behind the NAT.

Teliax
www.teliax.com

You can set a Name but it will only pass on Teliax to Teliax calls. Not to the PSTN. SIP Address is your device username @ which ever proxy Teliax proxy you want to register to (DEN, ATL, NYC, LAX). Registry Server is whichever proxy you are registering to (make sure you have it set in the Numbers section of the Teliax dashboard in order to receive inbound calls). User Name is your device name from the Devices section of your account.

Domain: den.teliax.net (see notes)
User Name: your device username
